# Bowling Layout Adapter

A web-based tool for bowling layout conversion and PAP (Positive Axis Point) adjustment. This tool helps pro shop operators and bowlers convert between different layout systems (Dual Angle, VLS, 2LS) and recalculate layouts when a bowler's PAP changes.

## Features

- **Layout Converter**: Convert layouts between:
  - Dual Angle
  - VLS (Vector Layout System)
  - 2LS (2-Handed Layout System)
- **PAP Adjuster**: Recalculate an existing layout for a new PAP to maintain the same ball reaction.
- **Reactive Interface**: Real-time calculations as you type.
- **Bilingual Support**: Full support for English and Korean (한국어).

## How to Use

1.  Open `index.html` in any modern web browser.
2.  **Converter**: Select your source system, enter the values, and see the converted values for your target system immediately.
3.  **PAP Adjuster**: Enter the bowler's old PAP, the layout on the ball, and the new PAP. The tool will provide the adjusted layout parameters.
